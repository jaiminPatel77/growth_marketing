# Growth Marketing Service

A modern, high-conversion portfolio website for a B2B digital marketing agency. Built with Angular 13 and Tailwind CSS.

## Features

- **Modern UI**: Clean, responsive, and professional design.
- **Tailwind CSS**: Utility-first styling with custom themes and dark mode support.
- **Conversion Optimized**: Lead generation forms and clear CTAs.
- **Service & Case Study Pages**: Dedicated sections to highlight marketing offerings and past successes.

## Development Server

1. Run `npm install` to install dependencies.
2. Run `ng serve` or `npm start` for a dev server.
3. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## Build

Run `ng build` to build the project. The build artifacts will be stored in the `dist/growth-marketing-agency` directory. This is the folder you use for deployment (e.g., to Vercel or Netlify).

## Deployment (Vercel/Netlify Recommended)

This project is perfectly suited for zero-config deployments on Vercel or Netlify.

- Connect your GitHub repository to either platform.
- The build command should automatically default to `npm run build` or `ng build`.
- The output directory will be `dist/growth-marketing-agency`.
